# Python Assignment Solutions

This repository contains Python solutions for three different tasks demonstrating basic programming concepts such as prime number checking, finding the largest number in a list, and building a simple calculator.

---

## Task 1: Prime Number Checker

**Description:**  
This script checks whether a given number is a prime number.

**Implementation Details:**
- Function `ip(number)` returns `True` if the number is prime, otherwise `False`.
- Numbers less than or equal to 1 are considered not prime.
- Uses a loop up to the square root of the number for efficient checking.
- Prompts the user to enter a number and prints whether it is prime.

## Task 2: Find the Largest Number in a List

**Description:**  
This script finds the largest number in a list provided by the user.

**Implementation Details:**
- Function `fl(numbers)` takes a list of numbers and returns the largest.
- Returns `None` if the list is empty.
- Iterates through the list and compares each number to determine the largest.
- Prompts the user to enter numbers separated by spaces and prints the largest number.
